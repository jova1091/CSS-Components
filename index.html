Modales (Nativo <dialog>)

Componente modal accesible utilizando el elemento nativo del navegador. Soporta cierre con tecla ESC, foco atrapado y animaciones CSS.

Título del Modal

Este es un modal nativo. El navegador maneja automáticamente:

  • El foco del teclado (Focus Trap).
  • El cierre con la tecla Escape.
  • La accesibilidad (Roles ARIA).
Modal Pantalla Completa

Este modal utiliza la clase .modal-fullscreen que sobrescribe las variables CSS:

.modal-fullscreen {
  --modal-width: var(--modal-fullscreen-width);
  --modal-margin-x: var(--modal-fullscreen-margin-x);
  --modal-border-radius: var(--modal-fullscreen-border-radius);
}

Esto demuestra el poder de la arquitectura basada en variables.

Navbar Documentation

El componente Navbar es responsive y soporta navegación móvil tipo Offcanvas (Sidebar). A continuación se detallan las clases de alineación disponibles. Adicionalmente, el comportamiento 'sticky' (fijo arriba) es opcional y se activa añadiendo la clase .navbar-sticky al elemento nav.

Nota: Los siguientes ejemplos son estáticos para demostración. El Navbar principal del sitio está fijado arriba (Sticky).

Estructura Base


<nav class="navbar navbar-sticky">
  <div class="container-full">
    <!-- Logo -->
    <a class="navbar-brand" href="#">Brand</a>

    <!-- Botón Móvil -->
    <button class="navbar-toggler" ...>...</button>

    <!-- Contenido Colapsable -->
    <div class="navbar-collapse" id="menu">
      <!-- Menú de Navegación (Aquí va la clase de alineación) -->
      <ul class="navbar-nav navbar-nav-center">
        <li><a class="nav-link" href="#">Item</a></li>
      </ul>
      
      <!-- Otros elementos (Botones, etc) -->
      <div>...</div>
    </div>
  </div>
</nav>
